# OpenSpending Javascript libraries 

All minification assumes you have `uglifyjs` installed. If you don't, use

    npm install uglify-js

# Using openspendingjs

## OpenSpending Icons

openspendingjs comes with a few svg icons, e.g. used in the bubbletree and the dailybread. If you want to use these icons Yoshihide Jimbo has compiled [a great overview page](http://jmblog.github.io/openspending-icons/).
